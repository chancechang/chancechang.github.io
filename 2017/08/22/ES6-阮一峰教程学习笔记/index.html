ES6-阮一峰教程学习笔记

分章节记录重点内容与思想

第一章 babel环境配置

使用webStorm新建项目
安装babel

babel-cli

命令行转码：babel-cli
转码后结果输出到标准输出 命令行输入babel example.js
转码后结果输出到文件 命令行输入babel example.js --out-file compiled.js或者babel example.js -o compiled.js

整个目录转码

babel src --out-dir lib或者babel src -d lib

babel-node

命令行输入babel-node进入REPL环境,可以直接运行es6脚本，也可以输入命令运行babel-node example.js

babel-register

babel-register改写require命令，为它加上一个钩子，每当 require加载.js、.jsx、.es、.es6为后缀的文件都会自动进行转码
require("babel-register");之后就不用手动对require加载的文件转码了

babel-core

如果要调用babel的API进行转码，需要使用babel-core
var es6Code = 'let x = n => n + 1'; var es5Code = require('babel-core') .transform(es6Code, { presets: ['latest'] }) .code;

babel-polyfill

babel默认转换句法，而不转换新的API，如Iterator、Generator、set、maps、Proxy、Reflect、Symbol、Promise等全局对象，以及一些定义在全局对象上的方法
在脚本头部，加上
import 'babel-polyfill'; 或者 require('babel-polyfill');

let与const

let

块级作用域：在循环体内，设置循环变量的是父作用域，循环体内部是一个子作用域，两者有各自单独的作用域
不存在变量提升：声明的变量必须在声明后使用，否则报错（注：var声明但不赋值，let不声明，不赋值）
暂时性死区：区块对声明的变量形成封闭作用域，外部作用域对该变量的声明对其无影响，区块内变量声明完成前都是该变量的死区，
typeof未声明的变量不会报错，typeof位于死区内变量会报错
不允许重复声明：一个作用域只能声明一次
为什么需要块级作用域？
外层作用域与内层相互独立，外层作用域无法读取内层变量，内层可读取外层变量，内层作用域可定义外层作用域同名变量
块级作用域与函数声明：块级作用域内可以声明函数（必须在{}内，否则报错），类似于let声明，因此，在块级作用域外不可引用该函数，但是ES6浏览器仍看做var声明
function f() { console.log('I am outside!'); } (function () { if (false) { // 重复声明一次函数f function f() { console.log('I am inside!'); } } f(); }());
输出：报错
function f() { console.log('I am outside!'); } (function () { if (true) { // 重复声明一次函数f function f() { console.log('I am inside!'); } } f(); }());
输出：I am inside!
function f() { console.log('I am outside!'); } (function () { f(); }());
输出: I am outside!
结论：但是ES6浏览器仍把块内函数声明看做var声明，变量提升，因此最好使用函数表达式代替声明函数

do表达式
由于块级作用域内部是多个语句，且外部无法访问内部值，因此使用do表达式让作用域返回值，示例：
let x = do{ let t = 7; t*t+1; }
注意：这只是一个提案

const

const是一个常量，一旦声明就立即初始化，且不能改变，改变常量值会报错
与let一样，const只在块级作用域内有效，不存在变量提升，存在暂时性死区，不可重复声明
本质：const保证的不是值不能改动，而是地址不能改动，对于简单类型数据，值保存在变量指向的那个地址，等同于常量，对于复合类型数据（数组与对象），变量指向的内存地址，保存的只是一个指针，只保证指针是固定的，至于指向的数据结构是否发生变化无法控制
const foo = {}; foo.prop = 123; foo.prop // 123 foo=3//TypeError
const声明对象，对该对象添加属性可以成功，但不能对该对象赋值
若要求不能对对象添加删除属性，则需要将对象冻结const foo=Object.freeze({})，但仍可对已经存在的属性做出修改
将对象彻底冻结
var constantize = (obj) => { Object.freeze(obj); Object.keys(obj).forEach( (key, i) => { if ( typeof obj[key] === 'object' ) { constantize( obj[key] ); } }); };
es6声明对象6种方法，class import let const

顶层对象的属性：
window 浏览器环境下顶层对象
global node环境下顶层对象
es5中顶层对象属性与全局变量是等价的
es6开始顶层对象将逐渐与全局对象脱钩
let const class 声明的全局变量，不属于顶层对象的属性，为保持兼容性，var function依然属于
为了能在各种环境取到顶层对象，一般使用this，this也有一定局限性，如下：全局环境，this返回顶层对象，es6模块与node模块中，this返回当前模块
获取顶层对象的方法：
var getGlobal = function () { if (typeof self !== 'undefined') { return self; } if (typeof window !== 'undefined') { return window; } if (typeof global !== 'undefined') { return global; } throw new Error('unable to locate global object'); };

3.解构赋值

变量的解构赋值

es6按照一定模式，从数组和对象中提取值，对变量进行赋值，称为解构(Destructuring)
let [a,b,c] = [1,2,3]//从数组中提取值，按照对应位置，对变量赋值，本质属于“模式匹配”
右边值少于左边值，解构不成功，变量值等于undefined，右边值对于左边值，解构可以成功
只有数组转为对象以后具备Iterator，所以等号右边也必须是数组,只要某种数据结构具备Iterator，就可以采用数组方式解构赋值
对于Set结构也可以用数组的解构赋值
let [a,b,c] = new Set(['a','b','c'])

解构赋值允许指定默认值 let [a,b=3,c] = [1,2,3]
只有赋予变量的值严格等于undefined，才会使用默认值，null不严格等于undefined

对象的解构赋值

let {foo,bar} = {foo:"123",bar:67} 对象的属性次序是随意的，取值时变量名与属性名同名才能取到值
若不一致还想取到值，可写成let {bar:foo} = {bax:"123",bar:67} foo//67 前者是匹配的模式，后者才是变量，真正被赋值的是后者
若相对前者也赋值，可写成let {bar,bar:foo} = {bax:"123",bar:67} :前面都是模式而不是变量
对象的解构也可以使用默认值var {x: y = 3} = {x: 5};只有赋予变量的值严格等于undefined，才会使用默认值
解构不成功，变量值等于undefined
若提前声明，则在解构赋值时会把大括号看成代码块，因此要加上小括号let x;({x} = {x: 1});//必须加上括号，否则会把大括号当做代码块
数组本质是特殊的对象，可以将数组用于特殊的解构赋值示例： let arr = [1, 2, 3];let {0 : first, [arr.length - 1] : last} = arr;

字符串的解构赋值

let [a,b,c] = 'hello' a//h b//e
let {length:len} = 'hello' len//5

数组和布尔值的解构赋值

let {toString: s} = 123;
let {toString: s} = true;
解构赋值规则: 只要等号右边不是对象或数组，就先转换成对象，undefined和null无法转换成对象，所有等号右边=undefined或null会报错

函数参数的解构赋值

[[1, 2], [3, 4]].map(([a, b]) => a + b); // [ 3, 7 ]
区分：
function move({x = 0, y = 0} = {}) { return [x, y]; }
上文是对x,y指定默认值，move({})使用默认值，结果都为0
function move({x, y} = { x: 0, y: 0 }) { return [x, y]; }
上文是对move的参数使用默认值 move({})参数不为空，使用{},x,y结果都为undefined
[1, undefined, 3].map((x = 'yes') => x); undefined会触发函数参数默认值

圆括号问题

模式中最好不要出现圆括号
非声明语句的非模式部分可以使用圆括号[(b)] = [3]; // 正确 ({ p: (d) } = {}); // 正确

解构赋值的用途

1.交换变量的值 [x,y] = [y,x]
2.从函数返回多个值
3.函数参数的定义
4.提取json数据（本质：对象解构赋值）示例：
let jsonData = { id: 42, status: "OK", data: [867, 5309] }; let { id, status, data: number } = jsonData;
5.函数参数默认值
6.遍历map结构
let m = new Map(); m.set('first','hello'); m.set('second','liming'); for(let [key,value] of m){ console.log(key,value); }
7.输入模块的指定方法
const { SourceMapConsumer, SourceNode } = require("source-map");

4.字符串的扩展

codePointAt

codePointAt用来测试一个字符是由两个字节组成还是由四个字节组成
使用for...of循环，因为它会正确识别32位的UTF-16字符,
function is32Bit(c) { return c.codePointAt(0) > 0xFFFF; } is32Bit("𠮷") // true is32Bit("a") // false
默认为十进制，转化为16进制：ch.codePointAt(0).toString(16)

string.fromCodePoint()

作用上恰好与codePointAt相反
String.fromCodePoint(0x20BB7)// “𠮷”
如果该方法有多个参数，则被合并成一个字符串返回
注意:fromCodePoint方法定义在String对象上，而codePointAt方法定义在字符串的实例对象上。

字符串遍历接口

es6为字符串添加了遍历接口，可使用for···of循环遍历，这个遍历接口可以识别大于0xFFFF的码点

at() 替代 charAt()

require('babel-polyfill')//垫片库 console.log('𠮷'.at(0))

区分at与codePointAt at返回具体值’𠮷’，codePointAt返回十进制134071

normalize()

Unicode正规化,可以识别合成的字符，不能识别三个及其以上合成的字符 NFC标准等价合成 NFD标准等价分解
'\u004F\u030C'.normalize('NFC').length // 1 '\u004F\u030C'.normalize('NFD').length // 2

indexOf includes startsWith endsWith

都返回布尔值
includes startsWith endsWith都支持第二个参数，指定开始搜素位置，endsWith与前两个不同，它针对前n个字符
var s = 'Hello world!'; s.startsWith('world', 6) // true s.endsWith('Hello', 5) // true s.includes('Hello', 6) // false

repeat

返回一个新字符串,参数如果是小数，会向下取整，负数或infinite会报错,0-(-1),NAN会默认为0，如果是字符串会转化成数字
"x".repeat(3) //xxx

padStart padEnd

这是一个提案，先加垫片
require('babel-polyfill') console.log('x'.padStart(5, 'ab') ) //ababx
第一个参数用来指定字符串最小长度，第二个用来补全的字符串，若原字符大于最小长度，则返回原字符串
常见用途'09-12'.padStart(10, 'YYYY-MM-DD')

模板字符串``

模板字符串可嵌入变量 ${} {}内部可以放入任意js表达式，可用来运算，以及引用对象属性，还能调用函数，但不能使用未声明的变量
如果{}中最终结果不是字符串，要转化成字符串，{}中放入未声明字符串会报错
模板字符串中若表示多行字符串，可保留空格和缩进，可用trim()来消除

标签模板

String.row()

模板字符串的处理函数，对应于替换变量后的模板字符串

5.正则的扩展

6.数值的扩展(Number)

es6二进制数值以0b开头，八进制数值以0o开头
将二进制、八进制转为十进制方法：Number('0b111')

Number.isFinite() Number.isNaN()

Number.isFinite(14)返回布尔值，检查数值是否有限，如果是字符串以及其他类型，直接返回FALSE
Number.isNaN(NaN)只对NaN以及包含NaN的运算返回TRUE，其他一律返回FALSE
isFinite(25) // true isFinite("25") // true Number.isFinite(25) // true Number.isFinite("25") // false isNaN(NaN) // true isNaN("NaN") // true Number.isNaN(NaN) // true Number.isNaN("NaN") // false Number.isNaN(1) // false

Number.parseInt() Number.parseFloat()

用法不变Number.parseInt === parseInt // true

Number.isInteger()

在js内部，整数和浮点数是同一个值，所以3与3.0表示的都是3，3===3.0 //true,所以3.0也是整数
注意：如果是字符串以及其他类型，直接返回FALSE Number.isInteger("15") // false

Number.EPSILON

是一个极小的常量，目的在于为浮点数之间运算设置一个误差范围

安全整数和Number.isSafeInteger

Number.MAX_SAFE_INTEGER
Number.MIN_SAFE_INTEGER
Number.isSafeInteger() 如果数值在最大值与最小值之间就返回TRUE,如果是字符串以及其他类型，直接返回FALSE
如果检验的是表达式，一定要检查参与运算的每个值，否则可能运算出错

Math.trunc()

Math.trunc()用于去除一个数的小数部分，返回整数部分
对于非数值类型会考虑转化成数值类型，再进行取整，转化之后为NaN或无法截取整数的值，返回NaN

Math.sign()

对于非数值类型会考虑转化成数值类型，再进行判断
正值 +1 赋值-1 0 0 -0 -0 其他值 NaN

Math.cbrt() 立方根

对于非数值类型会考虑转化成数值类型，再进行计算

Math.clz32()

返回一个数的32位无符号整数形式有多少个先导0
对于小数，该方法只考虑整数部分
对于非数值类型会考虑转化成数值类型，再进行计算
<<左移运算符与Math.clz32()方法直接相关
Math.clz32(0) // 32 Math.clz32(1) // 31 Math.clz32(1 << 1) // 30 Math.clz32(1 << 2) // 29 Math.clz32(1 << 29) // 2

Math.imul()

Math.imul(-2,1) //2
返回两个32位带符号整数相乘的结果,对于数值特别大超过js设置的精度的使用此方法才能返回正确的值

Math.fround()

返回一个数的单精度浮点数形式，更为准确

Math.hypot()

返回所有参数的平方和的平方根
对于非数值类型会考虑转化成数值类型，再进行计算

对数方法

双曲函数方法

Math.sinh(x) 返回x的双曲正弦（hyperbolic sine）
Math.cosh(x) 返回x的双曲余弦（hyperbolic cosine）
Math.tanh(x) 返回x的双曲正切（hyperbolic tangent）
Math.asinh(x) 返回x的反双曲正弦（inverse hyperbolic sine）
Math.acosh(x) 返回x的反双曲余弦（inverse hyperbolic cosine）
Math.atanh(x) 返回x的反双曲正切（inverse hyperbolic tangent）

指数运算符 **

2**3 === 8 //TRUE
Math.pow(a,b) 等同与 a**b 但在V8引擎中，两者运算方式有差异，计算较大的数值时，两者运算结果会有细微差异

Integer数据类型 提案

JavaScript的所有数字都保存成64位浮点数，这决定了整数的精确程度只能到53个二进制位
为了与Number相区分，Integer类型数据必须使用后缀n
Integer()与Number()转换规则基本一致
Integer() 对于无法转换成数值的字符串以及null、undefined 会报错 new Integer()会报错

除了不带符号的右移位运算符>>>和一元求正运算符+,其他能使用在Number的运算符都能使用在Integer
Integer()与Number()不能混合运算

7.函数的扩展

函数参数的默认值

function foo(x=1,y=2){}
参数变量是默认声明的，不需要使用let或者const再次声明
参数默认值是惰性求职，会把参数值直接代入函数体的参数中，不得不运算时才运算求解
设有默认值的参数一般为尾参数

函数的length属性

返回没有默认值的参数个数,rest参数不会计入这个属性，只计入设置了默认值之前的参数
(function (a) {}).length // 1
(function(...args) {}).length // 0
(function (a, b = 1, c) {}).length // 1
let x = 1; function f(y = x) { let x = 2; console.log(y); } f() // 1

设置某参数值不能省略

function throwIfMissing() { throw new Error('Missing parameter'); } function foo(mustBeProvided = throwIfMissing()) { return mustBeProvided; }

rest参数之后，不能再有其他参数
name属性，函数的函数名function foo() {}; foo.name // "foo"
(new Function).name // "anonymous"
bind返回的函数，name会加上bound前缀
function foo()(); foo.bind({}).name //bound foo

箭头函数

如果箭头函数代码块包含语句，需要用大括号括起来，如果返回对象，要用小括号把大括号括起来
注意：函数体内this对象就是定义生效时所在的对象，而不是使用时所在的对象，且this的指向固定不可变
不能做构造函数，与new一起使用会报错
不可以使用arguments
不能使用yield，因此不能使用Generator
